Resources & Tools

Through the innovation and development process, we built a number of open-source tools. These are available via the links below.

Climate-smart credit land management designs

These designs explain how sustainable farming practices can be integrated into credit terms in the dairy, tea, coffee, sugar and potato sectors. These systems, must overcome social, economic and institutional barriers to implementation, a list of which can be found here

OpenImpact Digital Monitoring Tool

This is a management information system (MIS) for geotagged photos verifying change of practices. The open-source code is available here. The system can be turned into an MIS for evidence of environmental actions attached to loan terms using this open-source code here. Code for an app for local review of data is available here.

Greenfi Community Eco-Credit Supporting Documentation

This includes the Greenfi community eco-credit white paper, open-source manual, and implementation checklist.

Overview of green financial instruments for self-help groups

Several organisations now develop green financing tools for informal financial self-help groups. This document tracks the opportunity and landscape.
